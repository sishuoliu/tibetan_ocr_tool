# Contributing / 贡献指南

Thank you for your interest in contributing to this project! / 感谢您对本项目的贡献！

## How to Contribute / 如何贡献

### Reporting Issues / 报告问题

If you find a bug or have a suggestion, please open an issue on GitHub:
- 如果发现 bug 或有建议，请在 GitHub 上提交 issue：
  - Describe the problem clearly / 清晰描述问题
  - Include steps to reproduce / 包含复现步骤
  - Provide error messages if any / 如有错误信息请提供

### Submitting Pull Requests / 提交 Pull Request

1. Fork the repository / Fork 仓库
2. Create a feature branch / 创建功能分支
3. Make your changes / 进行修改
4. Test your changes / 测试修改
5. Submit a pull request / 提交 Pull Request

### Code Style / 代码风格

- Follow PEP 8 for Python code / Python 代码遵循 PEP 8
- Add comments for complex logic / 为复杂逻辑添加注释
- Update documentation if needed / 如需要请更新文档

## Questions? / 有问题？

Feel free to open an issue for any questions or discussions.
如有任何问题或讨论，欢迎提交 issue。
